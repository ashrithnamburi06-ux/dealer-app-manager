# Capacitor Android Setup Instructions

## What's Already Done
- ✅ Installed @capacitor/core and @capacitor/cli
- ✅ Initialized Capacitor (capacitor.config.json created)
- ✅ App name: Dealrix
- ✅ App ID: com.dealrix.app

## Steps to Build Android APK/AAB

### 1. Install Android Platform
```bash
npm install @capacitor/android
npx cap add android
```

### 2. Build Your React App
```bash
npm run build
```

### 3. Sync Capacitor
```bash
npx cap sync android
```

### 4. Open Android Studio
```bash
npx cap open android
```

### 5. Build APK (for testing)
In Android Studio:
- Build → Build Bundle(s) / APK(s) → Build APK(s)
- APK will be in: `android/app/build/outputs/apk/debug/app-debug.apk`

### 6. Build AAB (for Play Store)
In Android Studio:
- Build → Generate Signed Bundle / APK
- Select "Android App Bundle"
- Follow signing wizard
- AAB will be in: `android/app/build/outputs/bundle/release/`

## Requirements
- Android Studio installed
- Java JDK 11 or higher
- Android SDK (API level 33+)
- Node.js installed

## Notes
- The app is already PWA-ready with manifest.json
- InstallButton component handles PWA installation on web
- For mobile, use the APK/AAB generated by Capacitor
- Update app icons in `android/app/src/main/res/mipmap-*` folders
